# 📘 Forms-FEM

## 🧩 Sobre o projeto

O Forms-FEM é uma aplicação web desenvolvida como solução de um desafio do Frontend Mentor, com foco na construção de formulários acessíveis, responsivos e com validação de dados utilizando JavaScript.

O desafio consistia em desenvolver um formulário de contato funcional, garantindo uma boa experiência de uso tanto para usuários comuns quanto para usuários que utilizam navegação por teclado e leitores de tela.

## 🎯 Objetivo do projeto

O projeto tem como foco:

- Praticar a construção de formulários acessíveis
- Reforçar conceitos de HTML semântico
- Trabalhar validação de formulários com JavaScript
- Melhorar organização estrutural do HTML
- Aplicar responsividade com Flexbox e CSS Grid
- Desenvolver interfaces adaptáveis para diferentes dispositivos
- Exercitar boas práticas de acessibilidade web

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📋 Funcionalidades principais

- ✅ Validação de formulário com JavaScript
- ✅ Verificação de campos obrigatórios
- ✅ Validação de formato de e-mail
- ✅ Mensagem de sucesso após envio
- ✅ Navegação completa via teclado
- ✅ Estrutura acessível para leitores de tela
- ✅ Layout responsivo
- ✅ Estados de hover e focus nos elementos interativos

## 🧠 Aprendizados

- Estruturação semântica de formulários HTML
- Organização de layouts sem prototipação prévia
- Manipulação e validação de dados com JavaScript
- Uso de Flexbox e CSS Grid para responsividade
- Implementação de estados interativos acessíveis
- Melhoria da experiência de navegação por teclado
- Construção de interfaces com foco em acessibilidade

## 💡 Nota
Este projeto foi importante para desenvolver uma visão mais estrutural sobre construção de interfaces, principalmente na organização semântica do HTML e na preocupação com acessibilidade. Além disso, reforçou conhecimentos sobre validação de formulários e responsividade em cenários mais próximos de desafios reais do front-end.
